Encrypt This!

You want to create secret messages which can be deciphered by the Decipher this! kata.
Encryption Rules

For each word in the message:

    The first letter must be converted to its ASCII code.
    The second letter must be switched/swapped with the last letter.

Input Format

A single line containing space-separated words representing the message.
Output Format

Print the encrypted secret message to standard output.
Constraints

    Message Length: Between 1 and 10,000 characters.
    Allowed Characters: Words contain only uppercase and lowercase English alphabets.
    Special Characters: No special characters in the input.

Sample Cases

Sample Input 1

Hello

Sample Output 1

72olle

Sample Input 2

good

Sample Output 2

103doo

Sample Input 3

hello world

Sample Output 3
